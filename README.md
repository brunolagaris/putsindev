## O que é o PutsInDev?

O **PutsInDev** não é apenas mais um blog de tecnologia. É um ecossistema colaborativo, gamificado e financeiramente justo para quem respira Ruby on Rails. O nome vem do clássico método `puts` do Ruby, mas também expressa aquela reação clássica de todo dev ao encontrar um bug bizarro (ou uma solução genial): *"Puts, agora sim!"*

### Por que colaborar?
- **Aprendizado Real:** Construído com o que há de mais moderno no ecossistema (Rails 8 + Inertia.js + React).
- **Gamificação:** Sistema de XP, Badges e Ranking de autores.
- **Fair Share:** Um modelo de doações via Stripe que realmente valoriza quem escreve.

---

## Tech Stack (The "Dev Power")

Para manter a elegância e a performance, usamos:

- **Language:** Ruby 3.4+ (The happiness language)
- **Framework:** Rails 8 (The magic one)
- **Frontend:** React + Inertia.js (SPA feel, Monolith heart)
- **Styling:** Tailwind CSS (Dark/Light mode native)
- **Database:** PostgreSQL (The reliable)
- **Container:** Docker (It works on my machine™)
- **Payments:** Stripe Connect (Split de doações)

## Funcionalidades "Fora da Caixinha"

### O Algoritmo de Split (40/60)
Valorizamos a consistência. As doações via Stripe são repartidas assim:
- **40%:** Manutenção da plataforma.
- **60%:** Autores do mês.
  - *Regra de Ouro:* Postou 5+ conteúdos no mês? Recebe o bônus cheio.
  - *Lembrou de postar?* Se ficar 1 mês em silêncio, recebe apenas a fração base e um "puxão de orelha" (reminder) automático para voltar a brilhar.

### Gamification
- **XP por postagem:** Ganhe níveis e mude a cor do seu perfil.
- **Badges:** De "Hello World" a "Ruby Sensei".
- **Multi-tenant:** Cada autor de elite pode ter seu próprio subdomínio/espaço personalizado.
